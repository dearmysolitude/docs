## 성능 데이터 모델링
- **성능 데이터 모델링이란 DB 성능 향상을 위한 사항이 데이터 모델링에 반영되도록 하는 것**
- 프로젝트 생명 주기에서 분석/설계 단계에서 시행하며, 성능 데이터 모델링 시점이 늦어질수록 재업무 비용이 증가한다.
